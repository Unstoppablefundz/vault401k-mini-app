# Vault401k – 401(k) Tracker Mini App

**Track your 401(k) balance, contributions, and growth — right inside Telegram!**

---

## Live Bot
[@Vault401k_bot](https://t.me/Vault401k_bot)  
**Tap "Open 401k" to start!**

---

## Features
- **Real-time balance**: See your 401(k) at a glance  
- **Add contributions**: Instantly update your balance  
- **10-year projection**: 7% avg return calculator  
- **Per-user data**: Saves automatically using Telegram ID  
- **No backend, no login, no cost**  
- Beautiful glass-morph UI

---

## How to Use
1. Open: [@Vault401k_bot](https://t.me/Vault401k_bot)
2. Tap **"Open 401k"**
3. Track, add, project — all in-chat!

---

## Tech Stack
- **Telegram Web Apps** (official)
- **Vercel** (free hosting)
- **HTML + CSS + JS** (no frameworks)
- **localStorage** (saves per user)
